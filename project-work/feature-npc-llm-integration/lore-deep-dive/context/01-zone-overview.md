# Zone Overview: Classic Through Luclin

This document catalogs all playable zones in the EQ server (expansions 0-3), with NPC population summaries.

## How to Read This Document

- **expansion**: 0=Classic, 1=Kunark, 2=Velious, 3=Luclin
- **ztype**: 0=outdoor dungeon, 1=indoor, 255=outdoor
- **unique_npc_count**: distinct NPC types that spawn in this zone
- **faction_variety**: number of distinct primary factions represented

## Zone Connections Map (Classic-Luclin)

These are the primary zone-to-zone connections (zone_points table). This defines travel routes and regional groupings.

### Antonica (Mainland) Region
- **West Commonlands** ↔ East Commonlands ↔ Kithicor Forest
- **West Commonlands** → Befallen (dungeon)
- **West Commonlands** → Nexus (Luclin spire)
- **East Commonlands** → West Freeport, Nektulos Forest, North Desert of Ro
- **Qeynos Hills** ↔ Blackburrow ↔ Everfrost Peaks ↔ Halas
- **South Qeynos** ↔ North Qeynos, Qeynos Hills, Qeynos Aqueduct
- **West Freeport** ↔ East Freeport, North Freeport, East Commonlands
- **Nektulos Forest** → Neriak (Foreign Quarter)
- **Lavastorm Mountains** → Solusek's Eye, Nagafen's Lair
- **North Desert of Ro** ↔ South Desert of Ro ↔ Oasis of Marr

### Faydwer Region
- **Butcherblock Mountains** ↔ Greater Faydark, South Kaladim, Dagnor's Cauldron, Ocean of Tears
- **Greater Faydark** ↔ Crushbone, Northern Felwithe, Lesser Faydark
- **Dagnor's Cauldron** → Kedge Keep (underwater), Estate of Unrest
- **Steamfont Mountains** ↔ Ak'Anon

### Odus Region
- **Toxxulia Forest** ↔ Erudin, The Hole
- **Erudin** ↔ Erudin Palace, Toxxulia Forest
- **Paineel** → The Hole (dungeon)

### Kunark Region
- **Field of Bone** → Kaesora, Lake of Ill Omen, Kurn's Tower
- **Lake of Ill Omen** → Cabilis (East), Warsliks Woods
- **Warsliks Woods** → Cabilis (West), Frontier Mountains
- **Frontier Mountains** → Burning Wood, Droga, Nurga
- **Burning Wood** → Skyfire Mountains
- **Trakanon's Teeth** → Sebilis, Old Sebilis
- **Overthere** → Chardok (via bridge)

### Velious Region
- **Iceclad Ocean** → Great Divide → Eastern Wastes, Western Wastes
- **Great Divide** → Thurgadin, Crystal Caverns
- **Eastern Wastes** → Kael Drakkel
- **Western Wastes** → Temple of Veeshan, Sleeper's Tomb
- **Cobaltscar** → Skyshrine

### Luclin Region (Moon of Luclin)
- **Nexus** — central hub zone; spires from Antonica and Kunark connect here
- **Bazaar** ↔ Shadow Haven ↔ Echo Caverns ↔ Paludal Caverns
- **Shadeweaver's Thicket** → Paludal Caverns, Hollowshade Moor
- **Hollowshade Moor** → Grimling Forest, Tenebrous Mountains
- **Tenebrous Mountains** → Maiden's Eye, Umbral Plains
- **Dawnshroud Peaks** → Netherbian Lair, The Deep
- **Scarlet Desert** → Akheva Ruins, The Grey
- **Katta Castellum** ↔ Sanctus Seru (rival city connection)
- **Shar Vahl** — Vah Shir city, isolated cluster with Shadeweaver/Paludal
